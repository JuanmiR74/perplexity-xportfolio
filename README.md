# FondoRadar v1

Scaffold inicial de la app de seguimiento de cartera de fondos y roboadvisors.

## Stack
- Vite + React + TypeScript
- Supabase Auth + Postgres + RLS
- Recharts para X-Ray
- Edge Function para refresco manual de precios

## Qué incluye
- Estructura de carpetas preparada
- Tipos TypeScript base
- Contexto de autenticación
- Layout principal con pestañas
- Stubs de vistas: Resumen, Fondos, X-Ray
- Esquema SQL inicial para Supabase
- Edge Function de ejemplo para refresco manual
- `.env.example`
